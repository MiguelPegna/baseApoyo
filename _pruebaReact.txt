Proyecto react vite
ejecutar en consola npm create vite@latest
despues y 
despues dar nombre del proyecto
Para este ejemplo usamos vanilla y javascript

instalar plugin react en la carpeta del proyecto
npm install @vitejs/plugin-react -E

instalar react en el package json
npm install react react-dom -E

Se crea un archivo vite.config.js para el punto de entrada de la aplicacion

import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
    plugins: [react()]
});


hay que cambiar el main.js por main.jsx porqeu el plugin instalado no esta preparado para cargar desde un js

Ahora que la app inicia
hay que instalar el linter
un linter se entiende como una herramienta a cargo de la mejora del código a través del análisis del código fuente, con el objetivo de identificar fallos o problemas en el sistema.
npm install standard -D

agregar en el package.json el sig codigo para habilitar el lint de standard
"eslintConfig": {
    "extends": "./node_modules/standard/eslintrc.json"
  }
