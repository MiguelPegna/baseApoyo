PROPS O PROPIEDADES
Es una forma de pasar variables o funciones de un componente a otro
El State o funciones que se crean en los compónentes solo estan disponibles en ese componente
LOs pomps son usasdos para evitar la duplicacion de codigo y reutilizar esas varibles, state o estado y funciones en otros componentes
Los Props se pasan del parent al children nunca de children a parent

SINTAXIS

<header
	nombreProp ={ data }
/>

Si se tiene un state que se va a pasar por diferentes componentes lo ideal es colocarlo en el archivo principal
Cada nidel de componentes debera tomar y pasar el prop hacía otros componentes. Tecnologías como Redux y Context evitan tener que hacerlo de esta forma
